## 跨地域备份

#### 跨地域备份流量空间费用
开启跨地域备份功能后，文件同步到目标地域会产生流量费用。不足1GB， 按1GB进行计费。 (单位: 元/GB)
| 可用区 | 单价 |
| -- | -- |
| 中国大陆 | 0.8 |
| 其他地域 | 1.2 |

#### 跨地域备份存储空间费用
开启跨地域备份功能后，已经同步到目标地域的备份文件，按照占用空间大小进行收费。每小时统计一次, 每天扣费一次，不足1GB按照1GB进行扣费。 (单位: 元/GB/天)
| 可用区 | 单价 |
| -- | -- |
| 中国大陆 | 0.008 |
| 其他地域 | 0.0092 |
